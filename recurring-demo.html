📊 How It Works

1. Type a task and check 🔁 to make it recurring.

2. Choose frequency: Daily / Mon-Fri / Pick specific days.

3. The habit auto-seeds into each matching day's task list.

4. Each day's completion is independent — check off today, it returns tomorrow.

5. Delete the rule from "My Habits" to stop it from recurring.

Try: Add "Daily current affairs" as Daily, then click different days in the calendar to see it appear!

Implementation Details

  • appState.recurringTasks[] stores rules with id, text, freq, days, startDate, endDate
  • seedRecurringTasks(dateStr) materializes rules into appState.tasks[dateStr] — idempotent via recurringId
  • Each seeded task has isRecurring: true and recurringId linking back to the rule
  • Completion is per-day (each day gets its own task instance with independent done state)
  • Deletion of a rule removes only future un-done instances; past completed ones are preserved
  • Integrates with existing rollover, Telegram digest, and progress stats — zero breaking changes
